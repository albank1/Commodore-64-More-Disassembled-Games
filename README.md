# Commodore-64-More-Disassembled-Games
Commodore 64 Games Disassembled so you can see the code and also add cheats

To create these I have used a combination of JC64dis, a Java disassemble available for the PC that can take most types of C64 files and disassemble.
Also, dasm an 8 bit assembler and cartconv a utility that comes with the VICE emulator.

The instructions on how to assemble and create files that you can run are in the header
